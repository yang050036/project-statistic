# project-statistic
一个基于 React + TypeScript + Node.js 的项目统计与标分质量分析系统。  
项目统计模块：数据列表展示、多条件筛选、任务数据导出（Excel/CSV）  
📈 统计分析模块：人员/项目/时间统计、环比同比分析、图表导出（PNG/JPEG/WebP）  
🧬 标分质量模块（可选）：CellRanger QC月度统计、实时MySQL读取、S1/V3.1和S3/V4分组统计  
技术栈：React 19 | TypeScript | Vite 6 | TailwindCSS 3 | Recharts | Node.js | Express
